# Pasta de PDFs de Prévia

Coloque aqui os arquivos PDF de prévia de cada material.
O nome do arquivo deve corresponder ao configurado no modo admin.

Exemplos:
- preview_precedentes.pdf
- preview_informativos.pdf
- preview_sentenca.pdf
- preview_dir_trabalho.pdf
- preview_proc_trabalho.pdf
- preview_constitucional.pdf
- preview_administrativo.pdf

Após fazer upload via modo admin (?admin=1), suba os PDFs
nesta pasta no GitHub e faça deploy no Netlify.
